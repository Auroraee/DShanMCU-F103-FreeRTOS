## 交互式题库

`quiz.html` 是一个基于 FreeRTOS 教程内容的交互式自测网页，包含 **100 道题**，覆盖第 7~19 章全部核心知识点。

**使用方式：** 直接用浏览器打开 `quiz.html` 即可，无需服务器。

**功能特性：**
- 4 种题型：单选、判断、多选、填空
- 按章节练习 + 综合测试模式
- 即时反馈与详细解析
- 答题进度持久化（localStorage）
- 错题回顾与成绩统计
